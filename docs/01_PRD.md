## 設計語言：2026 Casino Premium

博彩網站設計核心要素：

### 色彩系統

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg` | `#06080d` | 頁面深色背景 |
| `--surface` | `rgba(18,22,30,0.85)` | 半透明卡片底 |
| `--surface-hover` | `rgba(24,28,38,0.9)` | hover 加深 |
| `--gold` | `#d4a843` | 主金色 |
| `--gold-light` | `#f0c860` | 亮金色 |
| `--gold-gradient` | `linear-gradient(135deg, #d4a843, #f0c860)` | 按鈕/強調 |
| `--gold-glow` | `rgba(212,168,67,0.25)` | 發光陰影 |
| `--fg` | `#e8e6e1` | 主要文字 |
| `--fg-dim` | `#8a8884` | 次要文字 |
| `--border` | `rgba(212,168,67,0.12)` | 卡片邊框 |

### 卡片設計

- **玻璃擬態**：`background: rgba(18,22,30,0.85) + backdrop-filter: blur(12px) + border: 1px solid rgba(212,168,67,0.12)`
- **hover 動效**：`transform: translateY(-2px) + box-shadow: 0 8px 24px rgba(212,168,67,0.08)`
- **金邊裝飾**：卡片左側 2px 金色漸層邊線

### 頁籤設計

- 金線選中指示 `background: linear-gradient(90deg, transparent, var(--gold), transparent)`
- 未選中半透明文字
- badge 數字：金底黑字

### 按鈕

- 主按鈕：金色漸層底 + 深色文字 + hover 發光
- 標籤按鈕：膠囊形半透明底 + 細金邊
- 分頁按鈕：方形半透明 + hover 金邊

### 字體

- 標題：`'Cinzel', Georgia, serif`（飾線體，博彩感）
- 內文：`-apple-system, BlinkMacSystemFont, 'Segoe UI', system-ui, sans-serif`

### 動效

- 卡片 hover: 0.2s ease 上浮 + 陰影加深
- 頁籤切換: 0.15s 淡入淡出
- 燈箱開啟: 淡入 + scale(0.95→1)
- 標籤/按鈕: hover glow
