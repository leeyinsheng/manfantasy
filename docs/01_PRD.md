---

## 設計核心：APP Mode

不是網站，是一個「偽 APP」。桌面瀏覽器打開時，模擬手機 APP 的外觀和互動：

### APP 模式特徵

| 元素 | 網站模式 (v4) | APP 模式 (v5) |
|------|--------------|--------------|
| **容器** | 全寬 `max-w: 840px` 居中 | **手機尺寸** `max-w: 430px` 居中，外圍深色背景 |
| **頁籤** | 頂部 sticky tab bar | **底部導航列** (Bottom Tab Bar)，5 個圖示按鈕 |
| **Header** | 大標題 + 更新時間 | 緊湊頂欄，左側 app 名稱，右側更新時間 |
| **內容區** | 全寬 | 手機寬度，卡片化滾動 |
| **互動** | 滑鼠點擊、hover | 觸控優化（大點擊區域、無 hover 依賴） |
| **卡片** | hover 上浮 | **無 hover**，卡片點擊展開 |
| **捲動** | 瀏覽器原生 | 順滑滾動，底部導航常駐 |
| **邊距** | 網頁式邊距 | 手機式 padding（16px 安全區） |

### 移除的功能

| 功能 | 原因 |
|------|------|
| 搜尋列 (search-bar) | APP 模式不適合放搜尋框，改用下拉或省略 |
| 時間快捷按鈕 (time presets) | 同上，APP 以瀏覽為主 |
| hover 動效 | 手機沒有 hover |
| desktop 媒體查詢 | 僅保留手機寬度適配 |
